# Photobooth.co

Static marketing site for Photobooth.co with:

- Fullscreen video hero
- Service detail landing pages
- Embedded media galleries
- Formspree contact forms
- GitHub Pages-compatible static files

## Publish

1. Push this folder to a GitHub repository.
2. In GitHub, open `Settings` > `Pages`.
3. Set the site to deploy from the `main` branch root.
